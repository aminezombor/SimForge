# SimForge 0.1.1

SimForge is a Windows-first conversational robotics authoring workspace that keeps
Blender as scene truth and pairs every material change with plans, approvals, checkpoints,
deterministic validation, and verifiable USD or Isaac evidence.

Highlights:

- Guided, Balanced, and Autonomous action authority with permanent human gates
- Generated warehouse manipulator, licensed URDF path, and native-format staging
- Revision-stamped Blender inspection and deterministic geometry/robotics findings
- Portable modular USD export with hashes, reports, and OpenUSD reopen verification
- Optional Isaac Sim failure, approved Blender correction, and passing rerun lineage
- Sandboxed Electron renderer, protected credentials, narrow IPC, and no telemetry

Start with `SimForge-Setup.exe`. If Windows SmartScreen blocks the unsigned installer,
extract `SimForge-win32-x64-0.1.1.zip` and run `SimForge.exe`. Blender 4.5 LTS is required
and installed separately. NVIDIA NIM and Isaac Sim are optional.

See the repository README for the five-minute judge test, exact setup, limitations,
security, licensing, and uninstall instructions. Verify downloads with `SHA256SUMS.txt`.
