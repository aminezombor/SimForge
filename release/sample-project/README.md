# SimForge Warehouse Sample

This sanitized sample contains the generated warehouse mobile manipulator used for the
SimForge acceptance path. It includes the Blender source and the verified modular USD
package at `exports/verified-warehouse/scene.usda`.

Open `scene/project.blend` in Blender 4.5 LTS for visual inspection. Open
`exports/verified-warehouse/scene.usda` in an OpenUSD-compatible tool for package
inspection. The package includes its manifest, deterministic validation JSON, readiness
report, previews, source metadata, and third-party notices.

The project contains no API keys, credentials, global memory, private feedback IDs, or
development-machine paths. Physics values marked `ASSUMED` are demonstration metadata,
not measured hardware data.
